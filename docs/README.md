# AD-ArXiv-Daily

## 项目简介

这是一个自动化工具，用于每日追踪和整理自动驾驶(AD)领域最新的arXiv论文。本项目专注于自动驾驶相关领域，包括：
- 规划与控制 (Planning & Control)
- 预测 (Prediction) 
- 自动驾驶 (Autonomous Driving)
- 仿真 (Simulation)
- 感知 (Perception)

> 本项目改编自 [cv-arxiv-daily](https://github.com/Vincentqyw/cv-arxiv-daily)

## 特点
- 自动每日更新最新论文
- 按领域分类整理
- 包含论文标题、作者、arXiv链接和代码链接（如有）
- 支持自定义关键词和分类
